CCP4i2 Roadshow tutorials

A collection of short introductory tutorials

The following tutorials are an introduction to CCP4i2. They use a set of example data sets most of which which can be solved quickly.

Experimental Phasing

  • GerE A small structure with non-crystallographic symmetry that can be solved by either experimental phasing or molecular replacement. SAD and MAD data are available.
  • RNAse A small structure with non-crystallographic symmetry that can be solved by either experimental phasing or molecular replacement. SAD and SIRAS data are available.

Molecular Replacement

  • Beta-Blip A difficult molecular replacement problem.
  • CDK2/CyclinB/CKS2 A ternary complex MR problem.
  • CeuE A medium sized structure with non-crystallographic symmetry that can be solved by molecular replacement. Unmerged intensities are available.

Ligands

  • MDM2 MDM2 bound to the inhibitor nutlin 3A.
  • Glyco A a structure and reflection data of a fungal glycoprotein containing many extended N-glycans with some sugars in higher-energy conformations.
  • Fragment-soaked BAZ2B A ligand pipeline problem starting from an isomorphous structure.
